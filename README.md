# Harbour Light  
### A Guiding Platform for Migrants, Immigrants, and Refugees in India  

---

## Overview  

Harbour Light is a digital platform designed to assist immigrants, internal migrants, and refugees within India.  
It provides multilingual access to essential information and verified support resources, including legal assistance, employment opportunities, healthcare access, and local community guidance.  

The application is built with a responsive, app-like interface featuring a purple gradient design. It prioritizes clarity, accessibility, and user trust, ensuring that every visitor can navigate available support systems with ease and dignity.

---

## Features  

- Modern and adaptive interface optimized for mobile and desktop users  
- Support for six Indian languages: Kannada, Hindi, Tamil, Malayalam, Urdu, and English  
- Resource directory including verified helplines, shelter details, and employment links  
- Real-time multilingual translation  
- Progressive Web Application (PWA) functionality enabling limited offline access  
- Optional dark mode for user comfort  
- Location-based resource suggestions through geolocation  

---
